# W3x2Lni

《魔兽争霸III》地图的解包/打包工具。
将魔兽编辑器使用的二进制文件转为文本格式，以便于版本控制、搜索、批量编辑、对比、合并等操作。

## 说明

1. Obj 魔兽编辑器可以正常使用的格式，魔兽争霸III可以用但不是最佳选择。
2. Slk 魔兽争霸III可以正常使用的格式，魔兽编辑器无法打开。
3. Lni 魔兽争霸III和魔兽编辑器无法使用，但对版本管理工具友好的格式。

Obj和Lni之间可以无损转换。任意格式转为Slk都是有损转换，请只在给最终用户使用版本中使用。
